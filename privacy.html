Política de Privacidade

Última atualização: 20 de janeiro de 2026

1. Introdução

A SuperBusca - LinkedIn Job Automation ("nós", "nosso" ou "extensão") respeita sua privacidade e está comprometida em proteger seus dados pessoais. Esta Política de Privacidade explica como coletamos, usamos, armazenamos e protegemos suas informações ao usar nossa extensão do Chrome.

2. Informações que Coletamos

2.1 Informações Fornecidas por Você

  • Configurações da Extensão: Filtros de busca, palavras-chave, lista restritiva, preferências de idioma e outras configurações personalizadas
  • Credenciais do Google: Token de autenticação OAuth2 para acesso ao Google Sheets (apenas se você optar por usar esta funcionalidade)
  • ID da Planilha: ID da planilha do Google Sheets que você escolher para armazenar os dados

2.2 Informações Coletadas Automaticamente

  • Dados de Vagas do LinkedIn: Título, empresa, tipo de aplicação, link e descrição das vagas encontradas durante a busca
  • Dados de Uso Local: Informações sobre vagas processadas, ignoradas e aplicadas (armazenadas apenas localmente no seu navegador)

2.3 Informações que NÃO Coletamos

  • Credenciais do LinkedIn: Não coletamos, armazenamos ou transmitimos suas credenciais de login do LinkedIn
  • Dados Pessoais Sensíveis: Não coletamos informações como CPF, RG, dados bancários ou outras informações sensíveis
  • Histórico de Navegação: Não rastreamos sua navegação fora do LinkedIn
  • Dados de Terceiros: Não compartilhamos seus dados com terceiros para fins de marketing ou publicidade

3. Como Usamos Suas Informações

Usamos as informações coletadas exclusivamente para:

  • Funcionalidade da Extensão: Automatizar a busca de vagas no LinkedIn conforme suas configurações
  • Filtragem de Vagas: Aplicar filtros personalizados para encontrar vagas relevantes
  • Armazenamento de Dados: Salvar vagas encontradas no Google Sheets (se você optar por usar esta funcionalidade)
  • Deduplicação: Evitar processar a mesma vaga múltiplas vezes
  • Melhorar a Experiência: Manter suas preferências e configurações entre sessões

4. Armazenamento de Dados

4.1 Armazenamento Local

A maioria dos dados é armazenada localmente no seu navegador usando a API chrome.storage.local:

  • Configurações da extensão
  • Lista de vagas processadas
  • Cache de deduplicação
  • Preferências do usuário
Importante: Esses dados permanecem no seu dispositivo e não são transmitidos para servidores externos.

4.2 Armazenamento no Google Sheets

Se você optar por usar a integração com Google Sheets:

  • Os dados de vagas são enviados para a planilha do Google Sheets que você especificar
  • Você mantém controle total sobre essa planilha
  • Você pode deletar, modificar ou exportar os dados a qualquer momento
  • A extensão acessa a planilha usando OAuth2 com permissões limitadas apenas para leitura e escrita de planilhas

5. Compartilhamento de Dados

Não compartilhamos, vendemos ou alugamos seus dados pessoais para terceiros.

Os únicos serviços externos que podem ter acesso aos seus dados são:

  • Google Sheets API: Apenas se você optar por usar a funcionalidade de integração com Google Sheets
  • LinkedIn: A extensão interage com o LinkedIn apenas para buscar vagas públicas disponíveis na plataforma

6. Segurança dos Dados

Implementamos medidas de segurança para proteger suas informações:

  • OAuth2: Autenticação segura com Google usando o padrão OAuth2
  • HTTPS: Todas as comunicações com APIs externas são feitas via HTTPS
  • Armazenamento Local: Dados sensíveis são armazenados localmente no navegador, não em servidores externos
  • Permissões Mínimas: A extensão solicita apenas as permissões necessárias para funcionar

7. Seus Direitos

Você tem os seguintes direitos sobre seus dados:

  • Acesso: Visualizar todos os dados armazenados pela extensão
  • Modificação: Alterar suas configurações e preferências a qualquer momento
  • Exclusão: Deletar todos os dados armazenados desinstalando a extensão ou limpando o armazenamento local
  • Portabilidade: Exportar seus dados do Google Sheets a qualquer momento
  • Revogação: Revogar o acesso da extensão ao Google Sheets através das configurações da sua conta Google

8. Retenção de Dados

  • Dados Locais: Mantidos até que você desinstale a extensão ou limpe o armazenamento do navegador
  • Dados no Google Sheets: Mantidos indefinidamente até que você os delete manualmente da planilha
  • Token OAuth: Válido até que você revogue o acesso nas configurações da sua conta Google

9. Cookies e Tecnologias de Rastreamento

A extensão não utiliza cookies para rastreamento. A extensão interage com cookies do LinkedIn apenas para manter sua sessão ativa enquanto você usa o LinkedIn normalmente.

10. Alterações nesta Política

Podemos atualizar esta Política de Privacidade periodicamente. Quando fizermos alterações significativas:

  • Atualizaremos a data de "Última atualização" no topo deste documento
  • Notificaremos os usuários através de uma atualização da extensão
  • As alterações entrarão em vigor imediatamente após a publicação

11. Conformidade com Leis

Esta extensão está em conformidade com:

  • LGPD (Lei Geral de Proteção de Dados - Brasil)
  • GDPR (General Data Protection Regulation - União Europeia)
  • Políticas do Chrome Web Store

12. Uso por Menores de Idade

A extensão não é destinada a menores de 18 anos. Não coletamos intencionalmente dados de menores de idade.
